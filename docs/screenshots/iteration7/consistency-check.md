# 迭代7 UI一致性检查

## 页面和功能概览

### 统计页面 (StatsScreen)

#### 页面结构
- 顶部：分页选择器（支出/收入/资产）
- 时间选择器：日期显示 + 日/月/年切换
- 汇总卡片：收入/支出/结余
- 内容区域：根据Tab显示不同内容

#### 已实现功能

1. **Tab切换**
   - 支出Tab：显示收支趋势图 + 支出分类统计
   - 收入Tab：显示收支趋势图 + 收入分类统计
   - 资产Tab：显示资产负债统计 + 账户余额列表

2. **时间维度切换**
   - 日视图：单日收支趋势
   - 月视图：当月每日收支趋势
   - 年视图：数据按年聚合（趋势图隐藏）
   - 日期选择器：可切换年/月

3. **收支趋势图**
   - 绑定实际数据，显示净收支折线图
   - 蓝色线条 + 数据点
   - Y轴标签显示金额
   - X轴标签显示日期

4. **分类占比图**
   - 饼图显示各分类占比
   - 右侧列表显示详细数据
   - 显示分类名称、金额、百分比

5. **资产负债统计**
   - 总资产、总负债、净资产
   - 颜色区分：收入绿色、负债红色

6. **账户余额列表**
   - 按余额排序
   - 显示账户名称、类型、余额
   - 资产和负债分类显示

## UI一致性检查

### 颜色使用
- Primary: #3F82F6（主按钮、选中态、图表线）
- Income: #35C785（收入相关）
- Expense: #FF5A5F（支出相关）
- Background: #F6F8FC（页面背景）
- Surface: #FCFDFF（卡片背景）
- 分类颜色：使用项目定义的分类色

### 字体和字号
- 页面标题：18sp, SemiBold
- 模块标题：15sp, SemiBold
- 正文：14sp, Normal
- 辅助文字：12sp, Normal
- 金额：Monospace字体

### 间距和圆角
- 页面边距：16dp
- 模块间距：12dp
- 卡片内边距：16dp
- 卡片圆角：14dp
- 按钮圆角：10dp

### 组件使用
- KeacsCard：所有卡片容器
- SegmentedTabs：Tab选择器
- CategoryIcon：分类图标
- 数字键盘：未使用（底部固定按钮）
