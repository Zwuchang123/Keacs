# 迭代1页面与组件清单

## 页面

- 首页：本月结余、收入支出、快捷入口、最近记录空状态。
- 账单：搜索入口、月份栏、收入支出小计、账单空状态。
- 新增记录：类型切换、分类入口、金额区、账户日期备注、数字键盘、禁用保存态。
- 统计：统计类型切换、时间选择、趋势占位、排行空状态。
- 我的：分类管理、账户管理、导出备份、导入备份、设置、关于。

## 组件

- 页面外壳：`KeacsScaffold`、顶部栏、底部 5 项导航、中间新增按钮。
- 基础组件：概览卡片、快捷入口、分段切换、圆形图标、搜索框、空状态。
- 记账组件：金额显示、表单行、数字键盘、账单列表项。
- 菜单组件：分组菜单卡片、菜单行、分割线。

## 边界

- 本轮只做迭代1前端基础体验，不接入真实保存和数据库。
- 不显示登录、会员、云同步、联网入口和首版外功能。
