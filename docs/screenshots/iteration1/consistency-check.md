# 迭代1设计一致性检查

## 对照来源

- 设计图：`docs/screenshots/iteration1/design.png`
- 实现截图：`home.png`、`records.png`、`add-record.png`、`stats.png`、`mine.png`

## 检查结果

- 5 个主页面与设计图一致：首页、账单、新增记录、统计、我的。
- 视觉风格一致：浅色背景、蓝色主操作、白色卡片、圆形图标、轻阴影。
- 底部导航一致：固定 5 个入口，中间为蓝色圆形新增按钮。
- 空状态一致：首页、账单和统计页都保留空数据提示和“记一笔”入口。
- 新增页一致：支出默认选中，保存为禁用态，数字键盘完整显示。

## 说明

- 本轮只做迭代1前端基础体验，不接入真实业务数据。
- 设计图是总览稿，实际验收以单页截图逐页对照。
