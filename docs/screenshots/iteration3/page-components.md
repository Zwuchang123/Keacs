# 迭代3页面与组件清单

## 页面

- 分类管理：支出/收入分段、分类列表、停用标记、新增分类入口。
- 分类编辑：名称、方向二选一、图标选择、用于新建记录开关、删除和保存。
- 账户管理：净资产卡片、资产账户、负债账户、新增账户入口。
- 账户编辑：名称、账户性质二选一、账户类型、当前余额、用于新建记录开关、删除和保存。
- 状态示例：空名称、重复名称、删除规则、停用提示、保存失败。

## 组件

- 图标库：支出分类、收入分类、账户图标。
- 列表项：圆形图标、名称、状态说明、金额或箭头。
- 表单项：名称输入、金额输入、分段选择、开关。
- 操作区：底部删除按钮和保存按钮。
- 提示：错误文字、删除确认、停用说明。

## 边界

- 本轮只实现分类和账户管理。
- 账户增加/减少维护不属于迭代3，按确认设计未加入本轮功能。
