# 记账模块 PRD

## 1. 模块定位

记账模块是 Keacs 的首个核心 Capability Module，负责收入、支出、转账、分类、账户、统计、定时记账、备份导入和 AI 记账确认。

模块目标：

- 保持本地账本可信、离线可用。
- 为 Agent 提供可控的查账、记账、改账、删账和复盘工具。
- 验证模块可插拔、写入确认和公共组件复用框架。

## 2. 用户场景

- 快速记录一笔支出、收入或转账。
- 查看月度收支、结余、分类统计和账户余额。
- 通过自然语言查询、复盘或生成待确认记账操作。
- 导出、导入和迁移本地账本。

## 3. 功能范围

- 账目：新增、编辑、删除支出、收入和转账。
- 分类：收入分类、支出分类和账户分类管理。
- 账户：资产账户和负债账户管理。
- 统计：收入、支出、结余和分类统计。
- 定时记账：按周、月、年生成账目。
- 数据迁移：JSON 备份导出、合并导入和 Excel 添加收入/支出。
- Agent 工具：查账、消费复盘、生成记账草稿、生成修改草稿、生成删除草稿。

## 4. Agent 工具边界

- 查询类工具可以返回统计摘要、账目候选和数据范围说明。
- 写入类工具只能返回操作草稿，必须进入 ActionPreviewCard。
- 用户确认后，模块通过现有 UseCase 执行真实写入。
- 用户取消后，不写入任何数据。
- 删除和批量处理必须二次确认。

## 5. 复用公共组件

- Markdown 渲染：用于助手复盘、统计解释和帮助说明。
- AgentMessageCard：用于查账回答、错误和工具结果。
- ActionPreviewCard：用于新增、修改、删除和定时记账确认。
- AgentInputComposer：用于自然语言记账输入。
- 金额组件、趋势组件、列表组件、空态组件和摘要卡片。

## 6. 非目标

- 不做投资建议、借贷建议、保险建议或税务建议。
- 不让 Agent 绕过确认写入本地账本。
- 当前 Android 首版不做语音、图片、票据识别和文件上传式 AI 输入。

## 7. 验收标准

- 手动记账主路径稳定可用。
- 首页、统计和账户余额在写入后保持一致。
- AI 写入类操作 100% 经过确认卡片。
- 未启用在线助手时，本地记账完整可用且不上传账本数据。
- 模块文档与平台公共组件契约一致。
