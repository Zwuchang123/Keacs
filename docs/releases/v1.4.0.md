# v1.4.0

- 优化在线助手稳定性，模型响应慢或格式不稳定时，常见记账和查账会先使用本地规则给出可读结果。
- 优化助手长对话能力，支持保留历史对话、结合上下文继续追问，并在对话过长时提醒清理。
- 新增助手清空对话能力，用户退出后再进入仍保留原对话，同时保留引导式提问入口。
- 优化待确认操作卡片，直接提供“确认执行/确认删除”和“取消”按钮，预览内容改为用户可读的账目格式。
- 优化消息展示，消息卡片按内容自适应宽高，输入框支持多行自动换行，助手回复支持加粗和换行展示。
- 优化响应体验，放宽模型响应等待时间，发送中和回复后显示处理耗时，不再把慢响应直接当失败。
- 修复助手页和设置页输入法弹出时的界面抖动与遮挡问题。
- 优化官方后端兜底策略，模型调用异常时返回可读下一步，并继续保持日志脱敏。
