# v1.5.1

## 主要变化

- 修复助手流式输出偶发“模型服务不稳定/输出被截断/乱码”的体验问题：服务端不再因长回复触发 fallback，并为 SSE 明确声明 UTF-8。
- 思考过程改为可见的分阶段流式推进：模型等待期间持续输出进度步骤，不再长时间无输出。
- 思考过程与最终回复统一在同一条系统消息内连续呈现；需要确认的操作也会发送最终回复文本。
- 引导内容支持随时开关：开关按钮常驻放在“重新生成”按钮旁。
- 优化流式体感：细粒度分块 + 移除流式期间多余的“发送中”气泡，避免视觉割裂。
