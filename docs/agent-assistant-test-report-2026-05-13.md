# 助手对话测试报告

日期：2026-05-13

## 测试方法

本次采用可回放对话测试：用 150 条场景逐条请求助手接口，检查是否有可读回复、是否泄露 JSON 或内部信息、是否能在写入类任务中生成待确认预览。

## 初始问题复现

| 范围 | 结果 |
| --- | --- |
| 后端助手测试 | 27 通过，3 失败 |
| App 本地兜底测试 | 9 通过，3 失败 |

失败集中在：

- 统计问题返回“已收到”，没有真实账本摘要。
- 单候选改单只追问，不生成修改预览。
- 定时记账把“每月1号”的“1”误判为金额。
- App 断网兜底不支持删账、改单和定时记账。

## 修复后回归

| 范围 | 结果 |
| --- | --- |
| 后端助手测试 | 30 通过，0 失败 |
| 150 条对话回放 | 150 通过，0 失败 |
| Android Lint、编译、Debug 打包和单元测试 | 63 通过，0 失败 |
| Android 设备测试 | 6 通过，0 失败 |
| Release 构建 | 1 通过，0 失败 |

## 覆盖内容

- 新增支出、收入、转账。
- 本月、最近 7 天、最近 30 天、半年、一年、全部账单分析。
- 单笔修改、删除、批量操作提示。
- 每周、每月、每年定时记账。
- 长对话和复杂任务切换。
- 投资、借贷、保险、税务、医疗等边界问题。
- 确认卡片字段展示和底部弹窗修改入口。

## 结论

当前自动化验证范围内，助手对话主链路已恢复可用，写入类操作仍保持“先预览、再确认、后写入”的安全边界。
