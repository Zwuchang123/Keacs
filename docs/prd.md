# Keacs PRD

## 1. 产品定位

Keacs 是个人自用记账 App，目标是简单、轻量、便捷、智能。基础能力本地离线可用，AI 助手是可选在线能力，用于降低记账和查账成本。

核心问题：

- 用户希望快速记录收入、支出和转账，不想被复杂财务功能打断。
- 用户希望看懂自己的收支、账户余额和消费结构。
- 用户希望用自然语言完成记账、查账、改账和复盘，但写入账本必须可控。

产品目标：

- 打开后能快速完成一笔记账。
- 本地账本稳定保存，未启用在线助手时不依赖网络。
- AI 助手能理解常见记账语言，并在写入前让用户确认。
- 用户能通过统计、账户和备份能力掌握并迁移个人账本。

成功指标：

- 手动新增、编辑、删除账目主路径可稳定完成。
- 首页、统计、我的页展示的收支和余额一致。
- AI 写入类操作 100% 经过用户确认。
- 未启用在线助手时，不上传账本数据且离线功能完整可用。

## 2. 用户与场景

目标用户是个人记账用户，主要场景包括：

- 日常快速记录一笔支出、收入或转账。
- 查看本月收入、支出、结余和每日账单。
- 管理分类、账户和定时记账。
- 导出、导入或通过 Excel 添加历史账目。
- 通过 AI 助手用自然语言记账、查账、改账、删账和消费复盘。

## 3. 当前范围

基础记账能力：

- 首次欢迎引导。
- 首页月度概览和每日账单明细。
- 新增、编辑、删除支出、收入和转账。
- 收入分类、支出分类、账户分类管理。
- 资产账户、负债账户管理。
- 定时记账。
- 收入、支出、结余统计。
- 本地 JSON 备份导出与合并导入。
- Excel 添加收入和支出账目。
- 设置、关于和更新入口。

AI 助手能力：

- 底部一级“助手”入口。
- 在线助手开关、模型服务模式、自定义访问地址、API Key 和模型名配置。
- 对话式记账、查账、改账、删账、定时记账和消费复盘。
- 写入类操作预览、编辑、确认和取消。
- 动态引导词、回答反馈、个性化记忆和自我优化。
- 在线失败时的可读提示和本地记账不受影响。

## 4. 信息架构

底部导航固定为 5 个入口：

1. 首页
2. 助手
3. 新增
4. 统计
5. 我的

主要页面：

- 欢迎页
- 首页
- 助手页
- 新增/编辑记录页
- 账目详情页
- 统计页
- 我的页
- 分类管理和编辑页
- 账户管理和编辑页
- 定时记账页
- 设置页
- 关于页

“我的”页承接低频功能入口和原概览信息，包括分类管理、账户管理、定时记账、导入导出、Excel 添加、设置、关于、当月概览和资产负债概览。

## 5. 核心流程

首次使用：

1. 用户看到欢迎页。
2. 点击“开始记账”后写入本机标记。
3. 后续打开 App 直接进入首页。

手动记账：

1. 用户从“新增”进入记账页。
2. 选择支出、收入或转账。
3. 填写金额、分类、账户、日期和备注。
4. 保存后更新账户余额、首页和统计。

AI 助手：

1. 用户在设置页启用在线助手并完成必要配置。
2. 用户在助手页输入自然语言。
3. 查询类问题直接回答，并说明数据范围。
4. 写入类问题先展示预览，必要时追问或让用户选择候选账目。
5. 用户确认后才执行写入；用户取消时不写入。
6. 在线助手不可用时展示原因和下一步，不影响本地记账。

## 6. 功能需求

### 6.1 首页

- 展示选中月份、结余、收入、支出和每日账单明细。
- 月份只能选择当前或过去月份。
- 每日明细按日期分组，组内按时间倒序。
- 无记录时展示空态。
- 首页不展示总资产、总负债和净资产。

### 6.2 记账

- 支持支出、收入和转账。
- 金额必须大于 0，最多 2 位小数。
- 日期不能是未来时间。
- 支出和收入分类必填，账户可不选；如已设置默认账户，优先带出。
- 转账必须选择转出账户和转入账户，且两者不能相同。
- 编辑和删除记录后，账户余额和统计必须同步修正。

### 6.3 金额输入

- 支持数字、小数点、退格、`+` 和 `-` 连续计算。
- 输入总长度限制为 11 个字符。
- 表达式结果必须大于 0 才可保存。

### 6.4 分类

- 分类分为支出、收入和账户分类。
- 分类名称不能为空，同一方向下不允许重名，最多 4 个字。
- 分类支持新增、编辑、停用；无历史记录的分类可删除。
- 已被历史记录使用的分类不能删除，只能停用。
- 停用分类不用于新建记录，历史记录继续展示和统计。

### 6.5 账户

- 账户分为资产账户和负债账户。
- 账户名称不能为空，全局不允许重名。
- 资产账户余额通常为正数，负债账户余额按负数存储和展示。
- 账户余额通过编辑页直接修改。
- 已被历史记录使用的账户不能删除，只能停用。

### 6.6 定时记账

- 支持收入、支出和转账定时记账。
- 支持按周、月、年重复。
- 到达生成时间后，本地生成对应账目并推进下次时间。
- 生成时间固定为 09:00。
- 分类或账户不可用导致生成失败时，停用该定时记账并提示原因。

### 6.7 统计

- 统计页分为支出、收入和结余。
- 支持按月和按年查看。
- 收入统计只计算收入记录，支出统计只计算支出记录，结余按收入减支出计算。
- 转账不计入收入、支出和结余统计。
- 已停用分类的历史记录仍参与统计。

### 6.8 备份、导入和 Excel 添加

- JSON 备份导出包含分类、账户、记录、备份版本、导出时间、App 版本和余额符号规则。
- JSON 导入只支持合并导入，不覆盖现有数据，不做去重。
- 导入时重新生成本地 ID，并用临时 ID 恢复关联关系。
- 支持读取备份版本 1 和版本 2。
- Excel 添加支持 `.xlsx`，字段包括日期、收支类型、分类、账户、金额和备注。
- Excel 添加只支持收入和支出，不支持转账。
- Excel 行内日期、类型或金额错误时跳过该行，其余行继续处理。

### 6.9 设置

- 支持默认记账账户和默认记账类型。
- 支持在线助手启用开关。
- 支持官方免费模型服务和用户自定义模型服务。
- 自定义模型服务需要访问地址、API Key 和模型名。
- 自定义 API Key 只保存在本机，不上传到 Keacs 后端，不写入日志。
- 未启用在线助手时，不发起 Agent 网络请求。

### 6.10 AI 助手

- 首版只支持文字输入，不支持语音、图片、票据识别和文件上传。
- 输入不能为空，发送中禁止重复发送。
- 助手消息包括用户消息、文本回复、任务进度、上下文范围提示、候选选择、操作预览、执行结果和错误消息。
- 助手需要展示本次统计或分析的数据范围。
- 常见查账范围包括今天、昨天、本周、最近 7 天、最近 30 天、本月、上月、今年、去年和全部账单。
- 长任务需要展示用户可理解的阶段，例如正在理解、读取账本、生成结果、等待确认。
- 最近对话可保留，用户可手动清空。

### 6.11 AI 记账理解

- 助手需要识别类型、金额、日期、分类、账户和备注。
- 未说明日期时默认今天；日期不能为未来时间。
- 金额、类型或分类无法确定时必须追问。
- 收入和支出未说明账户时，优先使用默认账户；没有默认账户时可留空。
- 转账缺少转出或转入账户时必须追问。

### 6.12 AI 写入安全

- 所有新增、修改、删除、批量修改和定时记账操作，必须先展示预览。
- 预览必须展示操作类型、影响对象、金额、日期、分类、账户、备注、影响条数和风险提示。
- 用户确认后，应用按现有业务规则执行写入。
- 用户取消后，不写入任何数据。
- 每个待执行操作必须有一次性标识，已确认、已取消或已失败的操作不能再次执行。
- 删除和批量处理必须二次确认；影响范围不明确时必须追问。
- 模型或后端不能把未执行的写入伪装成“已保存”或“已删除”。

### 6.13 AI 反馈和动态引导

- 助手回复旁提供点赞、点踩和重新生成入口。
- 反馈用于优化回答质量、上下文选择、动态引导和用户习惯记忆。
- 动态引导词根据日期、账本、最近对话和用户偏好生成。
- 每次展示 2 到 4 条短句，不遮挡确认卡片和错误提示。
- 自我优化不能绕过写入确认，不能自动修改用户账本。

### 6.14 数据边界

- 基础账本数据默认只保存在本地设备。
- 未启用在线助手时，不上传任何账本数据。
- 启用官方在线助手后，允许上传并保存账本、对话、反馈、偏好、任务状态、模型输入输出、上下文摘要和优化分析结果。
- 单次模型调用必须按任务选择上下文，不能无控制地塞入全部历史。
- 设备原始标识上传前必须转换为不可逆摘要。
- 用户自定义 API Key 不上传到 Keacs 后端。

## 7. 非目标

- 不做账号登录和云同步。
- 不做多人账本、家庭账本或共享账本。
- 不做复杂预算体系、投资组合、资产持仓、账单日、免息期和手续费管理。
- 不做语音、图片、票据识别和文件上传式 AI 输入。
- 不让 AI 自动写入、自动删除或绕过用户确认。
- 不提供投资、借贷、保险、税务、法律、医疗等专业建议。
- 不承诺收益或预测收益。

## 8. 验收标准

- 用户可完成首次进入、手动新增、编辑、删除支出/收入/转账的主路径。
- 金额、日期、分类、账户等非法输入会阻止保存并给出可读提示。
- 新增、编辑、删除记录后，首页、统计和账户余额同步更新。
- 停用分类或账户后，历史记录仍可读，新增时不再使用。
- 定时记账到期后生成账目，并在失败时停用对应模板。
- JSON 导出和合并导入可完成换机迁移主路径。
- Excel 添加可跳过错误行，并添加其余合法收入/支出记录。
- 未启用在线助手时，助手页提示启用配置，本地记账不受影响。
- AI 查询类回答说明数据范围，写入类操作必须先展示预览。
- 用户取消 AI 写入后，本地账本不变化。
- 用户确认 AI 写入后，通过现有业务规则保存，并刷新相关数据。
- 自定义 API Key 不上传、不明文长时间展示、不写入日志。
