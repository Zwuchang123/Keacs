# Keacs 公共组件索引

## 1. 组件定位

公共组件层用于沉淀跨模块、跨 App 可复用的能力。组件服务于记账、待办、搜索/攻略、资产观察和未来独立 App，但不直接拥有业务数据，也不直接写入模块数据库。

公共组件必须说明：

- 输入：组件接收的数据、配置、权限和事件。
- 输出：组件产生的展示结果、用户动作或操作草稿。
- 状态：默认、加载、成功、空态、错误、禁用和权限受限。
- 错误：失败原因、用户可理解提示和可恢复动作。
- 权限：需要访问的系统权限及拒绝后的替代路径。
- 隐私：是否处理敏感数据、是否可离线、是否会触发网络请求。
- 复用场景：可在哪些模块或 App 中直接使用。

## 2. 文本组件

### Markdown 渲染

- 输入：Markdown 字符串、文本样式、链接点击回调。
- 输出：可读的富文本内容。
- 能力：标题、段落、列表、加粗、斜体、链接、代码块。
- 降级：表格在移动端降级为分组列表；解析失败时展示纯文本。
- 约束：不得执行 HTML、脚本或远程资源加载。
- 复用：Agent 回复、攻略摘要、资产观察说明、帮助文档。

### 富文本块

- 输入：段落、列表、引用、代码块等结构化块。
- 输出：统一排版的内容流。
- 约束：业务模块只提供内容，不控制组件内部间距和颜色。

## 3. 输入组件

### AgentInputComposer

- 输入：当前文本、发送状态、可用输入能力、占位文案。
- 输出：发送、停止、重试、清空、语音转文字结果回填。
- 状态：空输入、可发送、发送中、识别中、禁用、错误。
- 约束：只发出用户动作，不直接调用模型或写业务数据。

### 语音转文字入口

- 输入：麦克风权限状态、识别模式、语言偏好。
- 输出：转写文本、识别失败提示、权限拒绝提示。
- 权限：使用前必须说明麦克风用途；拒绝后保留文本输入。
- 隐私：默认不启用；在线识别必须明确告知可能上传语音。
- 复用：Agent 输入、快速记账、待办捕捉、搜索提问。

## 4. Agent 组件

### AgentChatSurface

- 输入：消息列表、任务阶段、动态引导词、底部输入配置。
- 输出：发送消息、选择候选、反馈、重试、确认或取消操作。
- 约束：不感知具体模块数据库，不展示内部提示词或原始工具参数。

### AgentMessageCard

- 类型：用户消息、助手回复、错误消息、工具结果、系统状态。
- 能力：Markdown 渲染、长文本折行、反馈按钮、复制、重新生成。
- 约束：反馈按钮只出现在助手回复；错误消息必须说明下一步。

### ActionPreviewCard

- 输入：操作草稿、字段 Schema、风险等级、一次性操作 ID。
- 输出：确认、取消、编辑字段。
- 约束：不能直接执行写入；确认后由模块 UseCase 或等价业务入口执行。
- 复用：记账写入、待办创建、攻略收藏、资产观察提醒。

### TaskStageCard

- 输入：阶段名称、阶段说明、是否可取消。
- 输出：取消或等待。
- 约束：阶段文案必须面向用户，不暴露状态图节点名。

### SuggestionChips

- 输入：2 到 4 条短建议。
- 输出：选中建议并填入或发送。
- 约束：不能遮挡确认卡片、错误提示或主要输入区域。

## 5. 平台服务组件

- 权限门禁：统一解释麦克风、文件、照片、位置等权限用途和替代路径。
- 文件/分享入口：用于导入、导出、攻略保存和模块间分享。
- 密钥配置：用于模型服务、搜索服务或其他外部服务的本机密钥配置。
- 错误提示：统一展示网络失败、权限拒绝、解析失败和本地校验失败。

## 6. 可视化组件

- 金额组件：统一金额格式、正负号、收入支出颜色和等宽数字。
- 趋势组件：用于收支、待办完成、资产观察等轻量趋势。
- 列表组件：统一图标、标题、副标题、尾部数值和点击状态。
- 空态组件：短标题、短说明、可选操作，不堆叠长解释。
- 摘要卡片：展示模块摘要、数据范围和更新时间。

## 7. 组件验收

- 组件能在至少两个模块场景中复用，或明确属于平台基础能力。
- 组件不直接依赖模块 Repository、DAO、数据库或后端内部结构。
- 涉及写入时只产生操作草稿、事件或确认结果。
- 涉及敏感权限时，有用途说明、拒绝状态和替代路径。
- 大字体和 360dp 宽度下不遮挡、不溢出、不横向滚动。
